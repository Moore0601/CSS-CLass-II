Introduction to CSS

CSS is short for Cascading Style sheet, can be define as a styling language. It is used to control the visual presentation of webpage. CSS allow web developer to create visual appealing and responsive website, manage layouts, typograpgy, color schemes and animations.

Ways to write CSS

  1. Inline CSS
  2. Internal CSS
  3. External CSS

Inline CSS:

Inline CSS involve adding CSS directly within an HTML elements using style attributes. This method is typically used when you need to apply a unique style to a single elements.

Internal CSS:

is defined with a style tag inside the head section of the HTML elements. This method is used when you want to style a single HTML document without affecting other pages.

External CSS:

This is the best way to write CSS. External CSS is define in a seperate CSS file and linked to a HTML document using the link tag. This method is ideal for styling multiple pages consistently.
